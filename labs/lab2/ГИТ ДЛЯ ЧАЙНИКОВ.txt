Для начала лабораторных работ нужно

1) Проверить привязку аккаунта к git 

открываете cmd и водите команду
git config --global user.name
git config --global user.email

проверяете ваши данные или нет, если нет после данной команды вводите свои 

пример 
git config --global user.name <Ваш логин гитхаба>
git config --global user.email <Ваша почта>

Папка с которой вы прокидываете проект НЕ ДОЛЖНА находится на сетевом диске Z



2)Проверить привязку в диспетчере учетных данных

Перейти в меню Пуск -> диспетчер учетных данных -> учетные данные Windows 

найти запись github или все что связано с этим  названием, раскрыть и посмотреть правильность данных, если там не вы, то просто удалить эту запись  


3) Проверить привязку в Pycharm

В IDE pycharm перейти в настройки (settings)-> version control -> Github

и  проверить правильность данных, если не вы то перепривязаться

нажать на + -> login via Github -> authorize with github 

3) Только после того как убедились в правильности всех привязок, начинать лабораторную работу
